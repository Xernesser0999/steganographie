## Utilisation :
L'application permet d'insérer un méssage caché dans une image.  
les fichier pris en compte sont :  
- jpeg / jpg

Deux bouton sont disponible :  

**Add a file**  
Ce bouton sert a ajouter une image. Après cela divers information vous seront donné a propos de l'image :  
- Sa taille en octet
- Sa taille en pixel
- Son nom

Si un message est détecté dans l'image selectionné il sera affiché dans une boite de dialogue.  

**Add a comment**  
Après avoir choisi une image vous pouvez ajouter un message caché dans celle-ci a l'aide du bouton.
